# README

To run app do following:
* Configure database in *config/database.yml*
* Install needed gems
  `$ bundle install`
* Create database
  `$ rails db:create db:migrate db:seed`
* Set up client
  `$ yarn --cwd client`
* Run app with
  `$ rake start`

## Well done!
Now go to *localhost:3001* and login with email: admin@example.com and password: *password*.
